# Strike for glory

## Description

This game is a 2D platform fighter game done by "Strike Team 4+2" as a school project in "Sunderby folkhögskola" in Boden, Sweden.

## Gameflow

Strike For Glory is a local multiplayer game where every player only has one health point. Once a player has been hit by an opponent's attack they are gone for that round. By default, a full match will feature multiple rounds and the player with the most won rounds in the end, will win the match.
